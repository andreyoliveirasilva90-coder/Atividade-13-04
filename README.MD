# 🚀 TaskFlow Pro

## 📌 Descrição
O TaskFlow Pro é uma aplicação web de gerenciamento de tarefas desenvolvida com HTML, CSS e JavaScript, com foco em experiência do usuário (UX) e funcionalidades modernas inspiradas em aplicações SaaS.

## 🎯 Objetivo
Aplicar conceitos de desenvolvimento front-end aliados ao uso estratégico de Inteligência Artificial por meio de Engenharia de Prompt.

## 🚀 Funcionalidades
- ✅ Adicionar tarefas
- 🗑️ Remover tarefas
- ✔️ Marcar como concluída
- 🔥 Definir prioridade (Alta, Média, Baixa)
- 📅 Definir data e hora
- 🔍 Buscar tarefas em tempo real
- 🎯 Filtros (Todas, Pendentes, Concluídas)
- ⏰ Identificação de tarefas atrasadas
- 🌙 Dark Mode com persistência

## 🧠 Engenharia de Prompt
Durante o desenvolvimento, foram utilizados prompts estruturados com:
- Persona
- Contexto
- Tarefa
- Formato

A IA foi utilizada como assistente no processo de desenvolvimento, sendo todas as respostas analisadas criticamente antes da implementação.
