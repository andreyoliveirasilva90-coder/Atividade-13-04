# 📝 To-Do List - Projeto com IA

## 📌 Descrição
Aplicação de lista de tarefas desenvolvida com HTML, CSS e JavaScript, com suporte a prioridades e modo escuro.

## 🚀 Funcionalidades
- Adicionar tarefas
- Definir prioridade
- Marcar como concluída
- Remover tarefas
- Dark Mode 🌙
- Interface moderna

## 🧠 Engenharia de Prompt
Foram utilizados prompts estruturados com:
- Persona
- Contexto
- Tarefa
- Formato

A IA foi utilizada como assistente, não substituindo o raciocínio do desenvolvedor.
